# CHECKLIST — <Framework Name>

> 本檔分兩半：「載入前自檢」與「任務結束覆盤」。
> 兩半都必須被使用——只在開頭用是自欺，只在結尾用是失憶。

---

## 載入前自檢（Use Before）

在宣告「我要用 `<framework-name>`」之前，逐題回答。每題都要嘛打勾，要嘛明確寫出「不適用」並解釋。

- [ ] **我讀完 `FRAMEWORK.md` 了**，不是只讀摘要。
- [ ] **我看過 `META.md` 的相容性欄位**，確認本框架對我這個 session 來說是相容的。
- [ ] **我確認本任務屬於「何時用」列表**，且不在「何時不用」列表中。
- [ ] **我評估了反模式**——本任務有沒有可能踩到其中一個？
- [ ] **我看了「與其他框架的關係」**——有沒有跟其他正在使用的框架衝突？
- [ ] **我準備好接受 Step 順序的約束**——不是挑步驟跳著做。
- [ ] **我有 `framework-version` 標頭**（從 `META.md` 抄），待會寫產出時會附上。

> 任一題沒過 → 不要硬上。回頭讀 `FRAMEWORK.md`、換框架、或拆任務。
