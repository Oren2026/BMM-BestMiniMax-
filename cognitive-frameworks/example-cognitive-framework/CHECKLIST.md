> 本檔分兩半：「載入前自檢」與「任務結束覆盤」。
> 兩半都必須被使用——只在開頭用是自欺，只在結尾用是失憶。

---

## 載入前自檢（Use Before）

在宣告「我要用分層任務執行」之前，逐題回答。

- [ ] **我讀完 `FRAMEWORK.md` 了**，不是只讀摘要。
- [ ] **我看過 `META.md` 的相容性欄位**，確認本框架對我這個 session 來說是相容的（agent-version >=1.0.0、detector-rules = 1）。
- [ ] **我確認本任務屬於「何時用」列表**：跨多輪、模糊空間、失敗成本中等以上、需 handoff。
- [ ] **我確認本任務不在「何時不用」列表**：不是單輪問答、不是純機械執行、不是純探索。
- [ ] **我評估了反模式**：本任務有沒有可能踩到「跳過決策直接動手」「規劃拆太細」「自證有效」？
- [ ] **我看了「與其他框架的關係」**：本框架是流程型，不會跟其他框架衝突。
- [ ] **我準備好接受 Step 順序的約束**：決策 → 規劃 → 執行 → 驗收，不跳步。
- [ ] **我有 `framework-version` 標頭**（`1.0.0`），待會寫產出時會附上。

> 任一題沒過 → 不要硬上。回頭讀 `FRAMEWORK.md`、換框架、或拆任務。
